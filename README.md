# FastAPI Text-to-Speech Starter

TTS demo using Deepgram with FastAPI.

## Quick Start
```bash
git clone --recurse-submodules https://github.com/deepgram-starters/fastapi-text-to-speech.git
cd fastapi-text-to-speech
make init
# Create .env with DEEPGRAM_API_KEY
make dev
```

Visit: http://localhost:8080
